# WP FFmpeg Thumbnail Generator

Automatically generates thumbnails from uploaded videos using FFmpeg in WordPress.

## Installation
1. Upload the plugin to `/wp-content/plugins/`.
2. Activate it through the 'Plugins' menu in WordPress.
3. Ensure FFmpeg is installed on your server.

## Usage
- Generates thumbnails automatically on video upload.
